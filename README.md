# Teknologi Mobile - UI/UX Aplikasi Pembelajaran dan Manajemen Tugas

# Deskripsi 
EduConnect adalah aplikasi mobile yang dirancang untuk membantu proses pembelajaran secara digital dengan menyediakan fitur manajemen tugas, forum komunitas, notifikasi, dan profil pengguna dalam satu platform terintegrasi.

Aplikasi ini memungkinkan pengguna untuk melihat tugas yang diberikan, mengunggah hasil pekerjaan, berdiskusi dengan pengguna lain melalui komunitas, serta memantau aktivitas pembelajaran secara lebih efektif.

Project ini dibuat sebagai bagian dari pembelajaran dan pengembangan portofolio di bidang Mobile Application Development, UI/UX Design, dan Sistem Informasi.
